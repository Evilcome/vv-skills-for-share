# Slide Outline Reference Template

Use this template as the default backbone when drafting a slide or PPT outline.

Adapt it to the user's context, but preserve the core hierarchy:

- title
- chapter
- section
- key points

## Full Template

```md
# [Presentation Title]

## Overview
- Goal: [What this deck is trying to achieve]
- Audience: [Who will watch or read it]
- Context: [Pitch / review / training / keynote / internal update / roadmap / other]
- Tone: [Executive / technical / educational / inspiring / persuasive / neutral]
- Estimated length: [e.g. 10 minutes / 15 slides]

## Chapter 1: [Title]
- Chapter goal: [What this chapter should accomplish]
- Suggested slides: [e.g. 2-3]

### 1.1 [Section Title]
- Key points:
```

## Outline Quality Checklist

Before finalizing, make sure:

- each chapter advances the story
- each section has concrete key points
- there is a visible opening-middle-closing arc
- the outline is usable for later slide generation
- missing source material is labeled instead of hidden
